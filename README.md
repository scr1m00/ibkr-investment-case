# ibkr-investment-case

Структурный анализ Interactive Brokers Group (NASDAQ: IBKR) за FY2024 — финансовая модель, инвестиционный кейс и интерактивный дашборд. Разбор разрыва между ростом консолидированной прибыли и прибыли акционера через холдинговую структуру IBG LLC.

## Ключевой инсайт

Net income available for common stockholders вырос на 25.8% — на 4.6 п.п. быстрее консолидированной прибыли IBG LLC (+21.2%). Разрыв объясняется структурными факторами (рост доли владения IBG, Inc. в IBG LLC, механика Tax Receivable Agreement), а не операционным результатом, который сам по себе объясняется ростом клиентской базы (DARTs +61%, accounts +30%), а не процентным циклом ФРС.

## Содержание

- `IBKR_Financial_Model.xlsx` — финансовая модель (12 листов): P&L, баланс, Cash Flow, факторный анализ, структурный разбор NCI/EPS, сценарный анализ ставки, сводная таблица, графики
- `IBKR_Investment_Case.pptx` — презентация на 8 слайдов
- `IBKR_Dashboard.html` — интерактивный дашборд (открывается в браузере, без установки)
- `IBKR_Raw_Data.xlsx` — исходные данные без обработки, для самостоятельного анализа
